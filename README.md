Мультиплатформенное (Win, Mac, Linux) десктопное приложение скачивания файлов. 


**Установка**

Установить Node.js (v18) и запустить в корневом каталоге проекта:

####  `npm install`


**Скрипты**

####  `npm start`
Запуск в режиме разработки

#### `npm run make`

Сборка приложения в папку `.\out\make`
